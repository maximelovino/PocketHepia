Access(id: String, user: User, room: Room, startDate: Calendar, endDate: Calendar?, startTime: Int, endTime: Int)

This is the Access RoomDB Entity

Parameters

id - The unique identifier, also the primary key

user - The user for which the access is given, this is an embedded document here, it is not really necessary to store it for now, because we only have our accesses... but it is future proof

room - The room for which the access is given, as an embedded document

startDate - The start Date for the access, will be converted to Unix Time when stored

endDate - The end Date for the access, or null, will be converted to Unix Time when stored

startTime - The start Time for the access during the day, its minutes past midnight value

endTime - The end Time for the access during the day, its minutes past midnight value
