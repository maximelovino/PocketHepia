Transaction(id: String, title: String, amount: Double, to: User?, from: User?, date: Calendar, adminCharge: Boolean, stripe: Boolean)

This is the Transaction RoomDB Entity

Parameters

id - The unique identifier, also the primary key

title - The title, or description, of the transaction

amount - The amount of the transaction, with the sign

to - The destination user, or null if the user didn't exist anymore, as an embedded document

from - The source user, or null if the user didn't exist anymore or external charge, as an embedded document

date - The Date for the transaction, will be converted to Unix Time when stored

adminCharge - Specifies if it is a balance charge by an admin

stripe - Specifies if it is a balance charge by Stripe
